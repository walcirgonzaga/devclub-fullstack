## Aula 09 — Publicações

> Quem posta aparece. Simples assim.

**Frequência:** 1 post por semana, mínimo.

**Tipos de post que funcionam:**
- "Terminei o módulo de [X] e aprendi [Y]" + print do projeto
- "Erro difícil que resolvi: [problema] → solução foi [solução]"
- "Diferença entre [conceito A] e [conceito B] em JavaScript"
- "Meu projeto [nome] está no ar! Stack: React + Node.js" + link

**Estrutura de post:**
```
Linha 1: gancho (começa com número, pergunta ou afirmação forte)
Linhas 2-5: conteúdo
Linha 6: chamada para ação ("Qual foi o erro mais difícil que já resolveu?")
```

## Aula 10 — SSI (Social Selling Index)

**O que é:** pontuação do LinkedIn de 0 a 100 que mede presença.
**Verificar em:** linkedin.com/sales/ssi

**Meta:** SSI acima de 50.

**Como aumentar o SSI:**
- Perfil 100% preenchido (foto, capa, headline, sobre, experiência)
- Postar conteúdo regularmente
- Conectar com pessoas (50+ conexões)
- Interagir com posts (curtir, comentar)
- Mensagens (InMail) — usar com moderação

## Aula 11 — Networking

**Estratégia:**
1. Conectar com todos os colegas do DevClub
2. Seguir instrutores, devs sêniors, CTOs de empresas de tech
3. Comentar posts relevantes de recrutadores e tech leads
4. Não ficar só lendo — interagir

**Meta primeiros 2 meses:** +50 conexões/semana.

**Mensagem de conexão (curta e direta):**
```
Oi [nome]! Estou em formação como dev full stack pelo DevClub
e adorei seu conteúdo sobre [assunto]. Posso te adicionar?
```

## Aula 12 — Recomendações

> Recomendações = prova social. Aumentam muito a credibilidade.

**Quem pedir:**
- Colegas do DevClub que trabalharam em projeto com você
- Instrutores ou mentores do curso
- Qualquer pessoa que possa falar sobre sua dedicação ou habilidade

**Quando pedir:** antes de candidatar às vagas.
**Mínimo:** 2 recomendações antes de começar as candidaturas.

**Como pedir:**
```
Oi [nome], estou organizando meu LinkedIn para candidatar às primeiras vagas.
Trabalhamos juntos em [contexto]. Você poderia escrever uma recomendação
rápida sobre minha dedicação ou o que desenvolvemos juntos?
```
